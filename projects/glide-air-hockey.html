GLIDE: Gamified Latency-Induced Dynamics in Teleoperated Air Hockey Environment

Project Overview

GLIDE investigates the impact of network latency on teleoperated systems using an innovative air hockey setup. The project utilizes two UFactory xArm 6 robotic arms and a touch-screen gameboard to visualize latency metrics and control adjustments induced by the vSTING module.

Key Features

  • Dual Robot Setup: Two UFactory xArm 6 robots for competitive air hockey gameplay
  • Touch-screen Interface: Interactive gameboard for user control
  • Latency Visualization: Real-time visualization of network latency metrics
  • Wireless Communication: Controlled degradation of wireless communication to study latency effects
  • vSTING Module: Integration for latency measurement and control adjustment

Research Objectives

  • Quantify the impact of network latency on teleoperation performance
  • Develop strategies for latency compensation in teleoperated systems
  • Create a gamified environment for studying human-robot interaction under latency constraints
  • Provide insights for applications in healthcare, autonomous vehicles, remote manufacturing, and rescue robotics

Technical Implementation

  • Robotic Arms: UFactory xArm 6 (x2)
  • Control System: Web-based controller with wireless communication
  • Latency Module: vSTING for latency measurement and visualization
  • Interface: Touch-screen gameboard for user interaction

Applications

The insights gained from GLIDE have direct applications in:

  • Healthcare: Teleoperated surgical robots and remote patient care
  • Autonomous Vehicles: Remote control and monitoring systems
  • Remote Manufacturing: Teleoperated industrial robots
  • Rescue Robotics: Remote operation in hazardous environments

Key Insights

GLIDE provides critical insights into latency management for teleoperated systems, helping researchers and engineers understand how network delays affect user performance and develop effective compensation strategies.
