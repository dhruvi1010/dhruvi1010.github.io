Reachy Robot Diagnosed for Teleoperation

Project Overview

This project focuses on diagnosing and developing Human-Robot Interaction (HRI) for the Reachy humanoid robot, with an emphasis on manipulation and teleoperation using Mixed Reality (MR). Reachy is an expressive, open-source humanoid platform ideal for HRI research. The work involves characterizing the robot for teleoperation scenarios and designing intuitive MR-based interfaces for remote control and manipulation tasks.

Key Features

  • HRI Diagnosis: Systematic evaluation of Reachy for teleoperation and manipulation use cases
  • Manipulation: Design and testing of manipulation workflows with the humanoid platform
  • Teleoperation: Remote control of Reachy with low latency and intuitive feedback
  • Mixed Reality Interface: MR-based control and visualization for immersive teleoperation

Technical Stack

  • Robot: Reachy humanoid (Pollen Robotics)
  • Middleware: ROS2
  • Interface: Mixed Reality (MR) for control and visualization
  • Software: Python, robot control and perception pipelines

Project Impact

The project contributes to HRI and teleoperation research by characterizing Reachy for remote manipulation and by combining it with Mixed Reality for more intuitive control. Outcomes support applications in assistive robotics, remote assistance, and human-robot collaboration.
