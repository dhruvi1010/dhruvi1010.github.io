Project Overview

Smart Pocket SOS is a emergency safety device designed to provide emergency assistance through GPS location tracking and GSM communication. When the panic button is pressed, the device automatically sends an SOS message with GPS coordinates to a preset emergency contact and initiates an emergency phone call. The device features a high-decibel buzzer for immediate attention and is powered by a Li-ion battery for portability.

Key Features

  • Emergency SOS Button: Single press triggers emergency response system
  • GPS Location Tracking: Real-time GPS coordinates sent via SMS with Google Maps link
  • Automatic Phone Call: Dials preset emergency contact number automatically
  • High Decibel Buzzer: Audible alarm for immediate attention in emergency situations
  • Compact & Portable: Pocket-sized wearable device design for easy carrying
  • Battery Powered: Li-ion battery for portable operation

Technical Implementation

  • Microcontroller: ATmega328P programming with Arduino IDE
  • GPS Integration: NEO-6M module for real-time location data acquisition
  • GSM Communication: SIM800L module for SMS transmission and voice calls
  • Power Management: Buck converter for efficient battery voltage regulation
  • PCB Design: Custom printed circuit board layout for compact integration
  • Software Libraries: TinyGPS++ for GPS data parsing, SoftwareSerial for communication

Project Objectives

  • Develop a cost-effective safety device for personal security
  • Integrate GPS and GSM technologies for emergency communication
  • Create a compact, wearable solution for vulnerable individuals
  • Provide real-time location sharing during emergencies
  • Enable quick emergency response through automated alerts

Applications

This safety device has applications in:

  • Personal Safety: Emergency assistance for individuals in threatening situations
  • Senior Care: Safety monitoring for elderly living alone
  • Healthcare: Medical emergency alerts and location tracking
  • Anti-aggression: Security alarms for businesses, jewelry stores, banks
  • Child Safety: Emergency communication device for children
