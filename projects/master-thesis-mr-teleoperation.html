AI-Enhanced Adaptive Guidance in Mixed Reality for Intuitive Teleoperation of Mobile Manipulators

Master's Thesis - Current Research Project

  1. Home
  2. Projects
  3. Master Thesis

Project Information

  • Category: Research Thesis
  • Status: Current (In Progress)
  • Institution: Technical University Dortmund
  • Robot Platform: Hello Robot Stretch 3
  • Technologies: ROS2, Unity, MoveIt2, Meta Quest 3

Project Overview

Developing an AI-assisted Mixed Reality (MR) teleoperation framework for the Hello Robot Stretch 3 that combines immersive interaction with adaptive autonomy. The system integrates a ROS2-Unity communication bridge, real-time motion planning (MoveIt2), and MR control via the Meta Quest 3 for intuitive head, hand, and joystick-based operation.

Key Features

  • ROS2-Unity Communication Bridge: Seamless integration between robotics middleware and MR environment
  • Real-time Motion Planning: Using MoveIt2 for smooth and safe trajectory execution
  • Multi-modal Control: Head, hand, and joystick-based teleoperation interfaces
  • AI-Driven Adaptive Guidance: Intelligent assistance modules for enhanced user experience

AI-Enhanced Features (Planned)

  • Reachable Workspace Visualization: Real-time visualization of robot's reachable space in MR
  • Collision Risk Alerts: AI-powered predictive warnings for potential collisions
  • Object-Aware Interaction: Context-aware assistance for object manipulation tasks
  • Adaptive Autonomy: Dynamic adjustment of assistance level based on user performance

Technical Stack

  • Robotics: ROS2, MoveIt2, Hello Robot Stretch 3
  • Mixed Reality: Unity, Meta Quest 3
  • AI/ML: PyTorch, Computer Vision, Motion Planning Algorithms
  • Communication: ROS2-Unity Bridge, Real-time Data Streaming

Research Impact

This research aims to enhance task efficiency and user safety in teleoperated robotic systems by combining the immersive capabilities of Mixed Reality with intelligent AI-driven guidance. The framework has applications in remote manufacturing, healthcare robotics, and disaster response scenarios.
