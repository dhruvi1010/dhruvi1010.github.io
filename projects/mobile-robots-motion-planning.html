Project Overview

This project focuses on autonomous robot navigation through path planning and optimization. The primary emphasis is on utilizing the Rapidly-exploring Random Tree (RRT) algorithm for global planning and the Timed Elastic Band (TEB) algorithm for local path optimization and obstacle avoidance.

Key Algorithms

RRT (Rapidly-exploring Random Tree)

  • Purpose: Global path planning
  • Function: Finds a feasible path from start to goal configuration
  • Advantages: Efficient in high-dimensional spaces, handles complex environments
  • Application: Initial path generation for mobile robots

TEB (Timed Elastic Band)

  • Purpose: Local path optimization and obstacle avoidance
  • Function: Optimizes the trajectory considering robot dynamics and obstacles
  • Advantages: Real-time optimization, considers velocity and acceleration constraints
  • Application: Smooth trajectory execution with dynamic obstacle avoidance

Project Features

  • Global Path Planning: RRT algorithm for finding initial feasible paths
  • Local Path Optimization: TEB algorithm for smooth trajectory execution
  • Obstacle Avoidance: Dynamic obstacle detection and avoidance
  • Real-time Navigation: Efficient algorithms for real-time robot control
  • Path Optimization: Continuous refinement of robot trajectory

Technical Implementation

  • Planning Framework: ROS-based navigation stack
  • Global Planner: RRT implementation
  • Local Planner: TEB algorithm
  • Sensor Integration: LIDAR, cameras for obstacle detection
  • Robot Platform: Compatible with various mobile robot platforms

Applications

This motion planning system has applications in:

  • Autonomous mobile robots
  • Warehouse automation
  • Service robotics
  • Autonomous vehicles
  • Search and rescue robots

Key Contributions

  • Integration of RRT and TEB algorithms for comprehensive path planning
  • Optimization of path planning for real-time performance
  • Robust obstacle avoidance in dynamic environments
  • Implementation suitable for various mobile robot platforms
